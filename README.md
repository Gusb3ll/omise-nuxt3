# omise-nuxt3

## [omise.js](https://www.omise.co/omise-js) integration with [nuxt 3](https://v3.nuxtjs.org)

### Files to look for

- nuxt.config.ts
- .env.sample
- pages/index.vue

## Setup

Install dependencies

```bash
yarn install
```

## Development

Start the development server on <http://localhost:3000>

```bash
yarn dev
```

## Production

Build the application for production

```bash
yarn build

yarn start
```
